# AI 智能深蹲计数系统

本项目是一个基于 **Python**, **PySide6 (Qt)**, **OpenCV**, **MediaPipe** 和 **pyttsx3** 开发的桌面端 AI 智能深蹲运动监测与计数系统。系统集成了安全的用户注册/登录、人脸识别防伪校验、实时全身与手势姿态检测、多组降序计数、组间休息倒计时、运动纠错和语音播报功能。

---

## 🚀 核心功能亮点

* **用户认证与人脸绑定**：支持中文、英文字母及数字的用户名注册，使用 SHA-256 哈希加盐加密保存密码，并在注册时采集人脸特征，登录时进行严格的比对校验。
* **多线程 AI 视频处理**：基于 `QThread` 运行 OpenCV 摄像头与 MediaPipe 姿态、手势检测，保证界面 UI 响应流畅，不卡顿。
* **智能姿态与手势控制**：
  * 全身入镜检测与侧身/正身姿态规范提示。
  * 支持通过手势（伸出 1 个手指表示开始/恢复，伸出 2 个手指表示暂停）控制运动进程。
* **多组降序计数与休息机制**：支持自定义总组数、每组目标个数及组间休息时间，单组完成后自动进入倒计时休息状态。
* **语音播报与智能节流**：通过多线程调用 `pyttsx3` 实现语音实时反馈，并内置防抖与冷却机制，避免高频刷新导致语音截断或重叠。
* **历史记录持久化**：运动结束后自动记录有效时长与总深蹲个数，保存在本地的 `user_data.json` 中。

---

## 🛠️ 所需运行环境

* **Python 版本**：Python 3.8 ~ 3.11 (推荐 3.10)
* **操作系统**：Windows 10/11 (部分语音库和摄像头依赖 Windows 平台原生支持)
* **硬件要求**：可用的一路标准 USB 摄像头或内置摄像头。
